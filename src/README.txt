Requirements:
- AlloLib
- Smartphone
	Transmits gyroscopes and accelerometers using App called “GyrOSC*” via OSC
	Transmission frequency : About 40Hz

AlloThresher is an AlloLib-based interactive granular synthesizer using a cellphone as a controller with gyroscopes and accelerometers. The gestural data is mapped into the parameter's of granular synthesizer. 

Error may occur if there is no proper OSC incoming data.


*GyrOSC, http://www.bitshapesoftware.com/instruments/gyrosc/
